# Milestone Plan

## Milestone 0 - Foundation

Deliverables:
- monorepo structure
- lint/format/test setup
- base docs
- CI skeleton
- local setup docs

## Milestone 1 - API domain and persistence

Deliverables:
- models
- migrations
- service layer skeleton
- test fixtures
- DB integrity rules

## Milestone 2 - Core API

Deliverables:
- round routes
- player routes
- course search and course assignment
- hole score routes
- contribution routes
- leaderboard and summary routes

## Milestone 3 - Web shell

Deliverables:
- route structure
- layout system
- query client
- API client wiring
- global error and empty states

## Milestone 4 - Round setup flow

Deliverables:
- new round screen
- course search and selection
- player setup
- transition into scoring flow

Deliverables:

- contribution controls

## Milestone 6 - Completion flow

Deliverables:
- leaderboard screen
- complete round action
- round summary screen
- locked-state UX

## Milestone 7 - Hardening

Deliverables:
- test expansion
- edge-case cleanup
- docs polish
- CI pass
- release checklist

## Milestone 8 - UX iteration

Deliverables:
- shell navigation and flow clarity improvements
- setup/scoring UX polish
- responsive treatment and visual consistency updates

## Milestone 9 - Authentication

Deliverables:
- user + session persistence model
- auth API endpoints and guards

- auth test coverage

## Milestone 10 - Invites and collaboration

Deliverables:
- round membership and invite model
- invite lifecycle endpoints (create/list/revoke/accept)
- collaboration UX in setup and summary
- role-based permission checks
